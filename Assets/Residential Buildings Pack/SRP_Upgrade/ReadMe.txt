Universal Render Pipeline upgrade
---------------------------------

Supported version: 7.1.8 and up only!

1) Import Residential Buildings pack to your URP project, and run Houses_URP unitypackage!
2) Enjoy!

High Definition Render Pipeline upgrade
---------------------------------------

Supported version: 7.1.8 and up only!

1) Import Residential buildings pack to your configured HDRP project!
2) Run the auto-upgrade tool on it (Edit/Render pipeline/project materials to HDRP)
3) When done, run and import Houses_HDRP unitypackage to overwrite shaders!
4) Set up volumes properly as per HDRP guidelines, and use volume profiles provided in HDRP_Resources folder.
5) Enjoy!
